# LocalMart – Quick Commerce Website

LocalMart is a full-stack e-commerce (quick commerce) web application built using Next.js, MongoDB, and NextAuth.

## Features
- User Registration
- User Login Authentication
- Multi-step Register Form
- MongoDB Database
- NextAuth Authentication
- JWT Based Login System
- Secure Password Hashing

## Tech Stack
- Frontend: Next.js, React, Tailwind CSS
- Backend: Next.js API Routes
- Database: MongoDB
- Authentication: NextAuth
